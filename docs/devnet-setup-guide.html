Conectar o feed TxLINE, obter as credenciais e por o backend pra rodar — passo a passo. VOCÊ FAZ = ação sua no navegador · AUTOMÁTICO = eu rodo o script.

0. Por que precisa de SOL (não é bug)

A doc oficial da TxODDS confirma: "Do I need SOL for the free tier? Yes. No TxL purchase is required, but the Solana wallet still needs SOL on the selected network to submit the on-chain subscription transaction and cover account rent." On devnet, request devnet SOL before starting."

Ou seja: "free" = sem pagar o token TxL, mas as credenciais nascem de uma transação on-chain (subscribe), e toda transação Solana custa uma taxa em SOL. No devnet o SOL é grátis (faucet). O airdrop da linha de comando só está temporariamente rate-limited — por isso usamos a faucet web (pinestake).

2. Obter as credenciais TxLINE AUTOMÁTICO

Com a wallet financiada, meu script de bootstrap faz o fluxo oficial (§ quickstart), 3 passos:

  1. Guest JWT — POST https://txline-dev.txodds.com/auth/guest/start
  2. Subscribe on-chain — subscribe(serviceLevel=1, weeks=4) no programa txoracle (devnet), assinado pela wallet de serviço. Devolve o txSig.
  3. Activate — assino a mensagem ${txSig}::${jwt} e chamo POST /api/token/activate → recebo o apiToken.

Resultado: TXLINE_JWT + TXLINE_API_TOKEN gravados no .env (nunca commitados). Você não precisa fazer nada aqui — só ter financiado a wallet no passo 1.

4. Rodar o backend AUTOMÁTICO

docker compose up -d          # Postgres local (já funciona)
pnpm migrate                  # cria as tabelas
pnpm dev                      # sobe a API + o ingester conecta no feed real

A partir daí: GET /api/feed/:matchId serve dados projetados do feed real, o ingester grava cada evento em feed_events, e o Swagger fica em /docs.

5. Settlement — a prize feature (Rust/CPI)

Duas camadas:

  • Simulação .view() — chama validate_stat do txoracle e lê o booleano contra o proof real. Funciona sem programa próprio. Construível assim que houver um proof (feito no passo 4).
  • CPI real (nosso programa Anchor) — o pote que faz o CPI e libera o prêmio no booleano. Pendente de destravar o toolchain anchor (mismatch de versão) — em andamento por mim, não depende de você.
